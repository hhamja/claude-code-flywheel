# 에스컬레이션 규칙 (.flywheel/BLOCKED.md)

같은 티켓에서 3회 실패했다면 접근이 아니라 **이해**가 틀렸을 가능성이 높다. 사람에게 넘겨라.
BLOCKED.md 가 존재하는 동안 루프는 결정적으로 정지한다. 사람이 해결 후 파일을 삭제하면 재개된다.

필수 형식:

```markdown
# BLOCKED

## 막힌 티켓
<티켓 경로>

## 시도한 접근과 결과
1. <접근> → <결과/에러 요지>
2. <접근> → <결과/에러 요지>

## 사람에게 묻는 질문
<정확히 1개 — 예/아니오 또는 선택지로 답할 수 있게 구체적으로>
```

나쁜 질문: "어떻게 할까요?" / 좋은 질문: "A 라이브러리 대신 B 로 교체해도 됩니까? (A 는 Node 18 미지원)"
